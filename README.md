# Awesome-Ai-library

Una lista curata di librerie, harness e skill eccezionali per lo sviluppo e l'orchestrazione di agenti AI.

## 🚀 Portale Web
Il progetto include una comoda interfaccia web per esplorare, filtrare e cercare rapidamente le librerie per categoria e tag. 
Basta aprire l'indirizzo principale o servire i file in locale per accedere al portale con un'interfaccia scura e moderna.

## 📚 Librerie Incluse

### Agent Harnesses & Frameworks
* **[awesome-agent-harness](https://github.com/Picrew/awesome-agent-harness)**: Curation list e risorse sull'infrastruttura deterministica (harness) per gli agenti.

### Agent Skills
* **[agent-skills](https://github.com/addyosmani/agent-skills)**: Set di skill e strumenti modulari pronti per assistenti AI.
* **[skills](https://github.com/mattpocock/skills)**: Action e moduli d'istruzione per automazioni e workflow agentici.
* **[ui-ux-pro-max-skill](https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)**: System prompt avanzato per guidare gli agenti nel design UI/UX premium.
* **[career-ops](https://github.com/santifer/career-ops)**: Pipeline e automazione AI per la gestione di carriera e ottimizzazione dei curricula.

### Codebase & Graph Analysis
* **[Understand-Anything](https://github.com/Egonex-AI/Understand-Anything)**: Mappatura interattiva dei codebase in grafi di conoscenza per l'analisi da parte degli agenti.
* **[code-review-graph](https://github.com/tirth8205/code-review-graph)**: Analisi statica e review del codice basata su grafi di relazioni.
* **[graphify](https://github.com/Graphify-Labs/graphify)**: Rappresentazione strutturata dei dati a grafo ottimizzata per sistemi AI.

### Optimization & Tooling
* **[caveman-compression](https://github.com/wilpel/caveman-compression)**: Compressione e ottimizzazione dei prompt/token per risparmiare fino al 65% del context.
* **[gstack](https://github.com/garrytan/gstack)**: Helper di esecuzione stack modulare per ottimizzare i workflow di sviluppo.
