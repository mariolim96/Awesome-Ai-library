# Gemini Project Instructions

Please remember the following guidelines when updating this repository:

1. **Language**: Keep the entire codebase, web portal, JSON data, and documentation (such as `README.md`) written in **English**.
2. **Generic Library Names**: If a library name is generic (e.g. `skills` or `agent-skills`), always reference the author/owner in the name or display title (e.g., `skills (by mattpocock)` or `agent-skills (by addyosmani)`).
